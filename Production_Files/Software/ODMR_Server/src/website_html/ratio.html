So funktioniert es

Der Ratio-Modus ermöglicht schnelle Magnetfeldmessungen, indem die Intensitäten an zwei Frequenzpunkten auf gegenüberliegenden Flanken eines ODMR-Dips verglichen werden. Das normalisierte Verhältnis r = (I₁-I₂)/(I₁+I₂) ist direkt proportional zur Verschiebung der Resonanzfrequenz und damit zum Magnetfeld.

Tipp: Wählen Sie f₁ und f₂ symmetrisch um das Minimum des ODMR-Dips für beste Empfindlichkeit.
